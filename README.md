# echoes

Handwritten learning notes.

This repo is only for learnings:

- what I understood
- what I tried
- what confused me
- what I can explain now
- what I want to revisit

No tweet drafts. No templates. No content system.

## Structure

```text
learnings/
  YYYY/
    MM/
      YYYY-MM-DD-topic.md
```

## Rule

Write the learning in my own words.

If I cannot explain it without notes, I should write the confusion honestly instead of pretending I learned it.
